Sentinel Prime Fire Truck

Sentinel Prime is a fire truck robotic system built around an Arduino Uno R3, programmed in C++ for motor control, servo actuation, and sensor integration. The system features an autonomous fire extinguishing mechanism with servos, a flame sensor, a recycled bottle as a water tank, and repurposed wood for the frame. It continuously scans for fires and automatically stops to spray water when one is detected, aiming to reduce human casualties in wildfires. The system uses SG90 servos for precise oscillation, a 4-channel relay for high-power DC motor control. Mounted on a remote control truck, the design combines power electronics, sensor feedback, and embedded control to enable autonomous firefighting.

Laser Tag System

An interactive laser tag game built using four BBC micro:bits and two custom 3D-printed blasters, programmed in Python. The system utilizes radio communication for real-time data exchange between micro:bits, enabling hit detection, health tracking, and gameplay synchronization. Each blaster is equipped with a trigger mechanism, an infrared (IR) emitter for shooting signals, and an onboard micro:bit for processing and transmitting data. The receiving micro:bits, acting as player health trackers, use LED matrix displays to indicate remaining health and game status. Additional features include sound effects for hits and eliminations, enhancing the experience.

Smart Home Automation System

A smart home monitoring and control system built using a BBC micro:bit, programmed in MicroPython. The system integrates multiple sensors, including a temperature and humidity sensor (e.g., DHT11, DHT22), a gas sensor (MQ series), and a motion sensor (PIR sensor) to continuously track environmental conditions. The micro:bit processes sensor data and communicates wirelessly using radio signals or Bluetooth, enabling real-time monitoring and remote control of smart home devices. The system can trigger automated responses, such as activating a fan when temperature rises, sending alerts for gas leaks, or turning on lights when motion is detected.

4-level bargraph display

Circuit Design
Multisim Simulation

The project involved designing, simulating, and implementing a 4-level LED bargraph display driver using comparators (LM311), LEDs,resistors, and a Zener diode for voltage reference. The circuit divides a stable reference voltage into four thresholds, each driving a corresponding LED to visually indicate input voltage levels. The design was tested and verified in Multisim by sweeping input voltages, checking LED response. The implemented circuit successfully displayed four distinct voltage levels and can be applied to audio level indicators, battery monitors, or sensor outputs.

This assignment taught me valuable skills in analog circuit design, simulation techniques, and troubleshooting.
