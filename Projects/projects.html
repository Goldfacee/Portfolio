    Sentinel Prime Fire Truck

    Sentinel Prime is a robotic firefighting system and my first ever solo project that I built around an Arduino and programmed in C++ for motor control, servo actuation, and sensor integration. The system features an autonomous fire-extinguishing mechanism with SG90 servos, a flame sensor, a recycled bottle used as a water tank, and a wooden frame. Mounted on a remote control truck, it scans for fire sources and automatically stops to spray water when one is detected. The project combines control, sensor feedback, relay-based motor switching, and practical mechanical integration to create a compact semi-autonomous response.

    Laser Tag System

    My team and I built an interactive laser tag game using four BBC micro:bits and two custom 3D-printed blasters, programmed in Python. The system uses radio communication for real-time data exchange, enabling hit detection and health tracking. Each blaster includes a trigger mechanism, an infrared emitter for shooting signals, and an onboard micro:bit for processing and communication. Receiving micro:bits act as health trackers and use LED matrix displays to show remaining health, while sound effects improve gameplay and feedback.

    Smart Home Automation System

    This smart home monitoring and control system was built using a BBC micro:bit and programmed in MicroPython. It integrates environmental sensors such as temperature and humidity sensing, gas detection, and motion detection to track conditions in real time. The micro:bit processes the incoming data and communicates wirelessly through radio or Bluetooth for remote monitoring and automation. The system can trigger practical responses such as turning on a fan, activating lights, or sending alerts when unsafe conditions are detected.

    Academic Work

    Here are some noteworthy class projects and assignments that I found especially interesting and technically rewarding.

    High Power PDB

    Circuit Design
    PCB Layout

    This project focused on developing a high-power power distribution board for general testing. The design involved PCB layout, component placement, power routing, and integration of control electronics for reliable current delivery. It strengthened my understanding of practical power electronics design, board-level integration, and the tradeoffs involved in routing higher-current paths while keeping the layout organized and manufacturable.

    Multistage Voltage Amplifier

    Circuit Design
    Implementation / Test Setup

    For this assignment, we designed and implemented a three-stage voltage amplifier with an intended overall gain of about 1000. The design used three cascaded NPN BJT stages, with the first two stages configured as common-emitter amplifiers for high gain and the final stage as a common-collector emitter follower for buffering and improved impedance matching. The project involved stage-by-stage troubleshooting, gain analysis, and comparison between simulation and measured behavior, highlighting the practical limitations of real analog circuits.

    Analog and Digital Voice Modulator

    Circuit
    System Build

    For our final project, we designed and implemented a multi-stage audio processing system capable of capturing, filtering, amplifying, and modulating a voice signal. The full architecture included a microphone input, pre-amplifier, band-pass filter, ESP32-based digital voice modulation stage, level display, power amplifier, and speaker output. This project required careful analog and digital interfacing, noise reduction, staged testing, and system-level troubleshooting to produce a working end-to-end voice-processing prototype.

    4-Level Bargraph Display

    Circuit Design
    Multisim Simulation

    This assignment involved designing, simulating, and implementing a four-level LED bargraph display using LM311 comparators, LEDs, resistors, and a Zener diode voltage reference. The circuit divided a stable reference into threshold regions, with each comparator driving an LED to indicate the input voltage level. The project reinforced my understanding of comparator-based analog design, voltage thresholding, and simulation-driven verification for indicator circuits.
